DCCells

Modeling dormancy in cancer cells

Download this project as a .zip file Download this project as a tar.gz file

This project aims to model the interactions of dormant cancer cells with the bone environment. The aim is to develop a agent-based multicellular model of the bone environment utilizing the framework FLAME. Later-on a more detailed implementation of specific signaling networks will be integrated running ODE-based models of them in COPASI and connecting those to the cell agents.
